### Composição química

Leitura do banco

``` r
df <- readxl::read_xlsx("data-raw/dadoscpfl.xlsx",
                        sheet = sheets[1],
                        skip = 1) %>% 
  rename(data = "...1") %>% 
  janitor::clean_names() %>% 
  mutate(
    amostra = str_to_lower(amostra),
    cd = as.numeric(ifelse(cd == "<1",0.5,cd)),
    cr = as.numeric(ifelse(cr == "<1",0.5,cr)),
    pb = as.numeric(ifelse(pb == "<1",0.5,pb))
  )
```
